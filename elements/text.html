      # TextElement

      文本元素， 继承 BaseElement

      # 参数

      属性名 描述 默认值
      text 文字
      color 字体颜色 #000000
      fontSize 字体大小 16
      lineHeight 行高 20
      textAlign 文字对齐方式 left
      textBaseline 文字基线 middle
      textDecoration 文本修饰, 参考css
      lineClamp 展示行数 1
