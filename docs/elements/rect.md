# RectElement

矩形元素， 继承 [BaseElement](base.md)

## 参数

| 属性名       | 描述                | 默认值 |
| ------------ | ------------------- | ------ |
| bgColor      | *背景颜色*          | #fff   |
| border       | *边框样式* like css |        |
| solid        | *是否实心*          | true   |
| borderWidth  | *边框宽*            | 0      |
| borderColor  | *边框颜色*          |        |
| borderRadius | *边框半径*          | 0      |
| boxShadow    | *阴影样式*          |        |
