# QRCodeElement

二维码元素 ， 继承 [BaseElement](base.md)

## 参数

| 属性名  | 描述             | 默认值 |
| ------- | ---------------- | ------ |
| content | *需要绘制的内容* |        |
| color   | *前景色*         | #fff   |
| bgColor | *背景色*         | #000   |
