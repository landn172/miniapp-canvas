# BaseElement

基本元素（不参与绘制）

## 参数

| 属性名 | 描述               | 默认值 | 类型   |
| ------ | ------------------ | ------ | ------ |
| width  | 元素宽             | 0      | number |
| height | 元素高             | 0      | number |
| top    | 距离canvas顶部距离 | 0      | number |
| left   | 距离canvas左边距离 | 0      | number |
